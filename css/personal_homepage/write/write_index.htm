钥匙掉进了厕所里

2006年10月23日，是一个沉痛的值得纪念的日子。就在这一天，我的钥匙串掉进了宿舍的厕所里。

我一直觉得自己是个运气不错的人。我的钥匙挂链快坏了，好几次都堪堪快从裤耳上脱落了，可它就是不掉，这更让我觉得自己运气不错。

可就在2006年10月23日这一天，我方便完后，整理衣服时，“呱棱儿”一声，我腰前的钥匙串掉到了那个深洞里。

一时间，我觉得自己运气并不是很好。

我去找涛哥。涛哥说：伸手下去掏，上次我把肥皂掉到我们厕所里，我就把它掏上来了。

回到宿舍，我脱掉上衣，趴在马桶边向下伸手。可是我的胳膊粗了点，手还没到底，但胳膊再也伸不下去。更糟的是，我想把胳膊抽出来也不是那么容易。我费了半天劲，骂了好几句他妈的，把胳膊捩得生疼，总算逃出了马桶的怀抱。

“涛哥，不行啊！我手伸不到底。”我跑到涛哥屋里。

“怎么可能呢？我上次一下就伸到底了。”涛哥说。

“我的胳膊粗，刚才险些就抽不出来了。”

涛哥捋起袖子，和我比了比胳膊，说：“的确是粗点。”

我说：“要不，你帮我去掏。”

涛哥说：“好，我去看看。”

回到我宿舍，涛哥也脱了上衣，来到马桶边伸手试了试，说：你们这个洞好像是小一点。

我心说：不小才怪，厕所又不打扫，洞边上都是厚厚的污垢。

涛哥伸手下去，脸都贴了地。我拿了几张纸给他垫在脸下。

“摸不到，好像不在正底下，是不是冲到旁边去了？”涛哥问。

“不会吧，我刚才小便完都没敢冲水。”

“哇，也就是说下面都是你的尿了？！”

“哈哈。”

涛哥真够意思，又伸手下去摸索。涛哥这个人平时不多言不多语，不懂世故，只爱玩游戏，到关键时刻却是最靠得住。

涛哥把手抽出来，在水龙头下冲，说：“不行，摸不到。干脆重新配一套吧。”

我仍不甘心，打电话给修水电的师傅，他让我把一个衣架撇直了做一个小勾碰碰运气。

我和涛哥依法试了试，用钩子在下面搅了搅，还是碰不到什么东西，也听不到什么响声，看来果然不在下面。

涛哥说：看来弄不上来了，重新配一套吧。

我还不甘心，因为我衣柜的钥匙也在那串钥匙里，如果钥匙上不来，我只能把衣柜的锁撬开了。

又把钩子放下去，搅了搅。

“看来我只能放弃了。”我边说边把钩子提上来。

咦？感觉不一样！我小心地往上提钩子。

“哈哈，我的钥匙在上面！涛哥，你看啊！”

涛哥也很高兴。

我打电话给我女朋友，把经过告诉了她。她说：“你要好好谢谢涛哥，换了别人还不一定帮你掏呢。”

是呀，知道我钥匙掉到厕所里的同学有好几个，可他们中还有谁肯向涛哥这样帮我呢。

涛哥走后，我拿出本本上网，遇到鹏，告诉他我的钥匙掉在厕所了。他问捞上来没有，我说我的胳膊粗，伸不下去。他说他要回来帮我掏。我赶紧告诉他已经钩上来了。鹏的回答早在我意料之中，他也是个好兄弟。

别的很多人，我想也想得出来，告诉他们之后，他们只会哈哈一笑，或者说些风凉话。
